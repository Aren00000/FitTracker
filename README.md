# Fit Tracker 💪 - Трекер тренировок и фитнеса

Приложение для отслеживания тренировок с анализом данных и визуализацией прогресса.

##  Требования

- .NET 8.0 SDK
- Windows (для WinForms)

##  Запуск проекта

```powershell
cd FitTracker
dotnet restore
dotnet run
```

##  Структура проекта

```
FitTracker/
├── Models/
│   ├── Exercise.cs      # Модель упражнения
│   ├── Workout.cs       # Модель тренировки
│   └── AppSettings.cs   # Модель настроек
├── Services/
│   ├── DataService.cs   # Работа с JSON/CSV файлами
│   └── AnalysisService.cs # Анализ данных
├── MainForm.cs          # Главная форма с логикой
├── MainForm.Designer.cs # Дизайнер формы
├── Program.cs           # Точка входа
└── FitTracker.csproj    # Файл проекта
```

##  Технологии

| Компонент | Технология |
|-----------|------------|
| GUI | WinForms (C#) |
| БД (настройки) | JSON |
| БД (тренировки) | CSV (с шифрованием) |
| Визуализация | Chart Control |
| Контроль версий | Git |

##  Функции

- ✅ **CRUD операции**: Добавление и удаление записей о тренировках
- ✅ **Валидация данных**: Проверка ввода пользователем
- ✅ **Шифрование данных**: XOR-шифрование CSV файла
- ✅ **Анализ данных**: Средний вес, максимум, минимум, общий объем
- ✅ **Визуализация**: График прогресса веса
- ✅ **Настройки**: Смена темы (светлая/темная), смена языка (RU/EN)

##  Формат данных

### settings.json
```json
{
  "Theme": "Light",
  "Language": "ru",
  "Unit": "kg"
}
```

### trainings.csv.enc (зашифрованный)
```
DATE|2024-01-15
EX|Жим лежа|80|10|4
EX|Присед|100|8|3
END
```

##  Шифрование

Данные тренировок шифруются простым XOR-алгоритмом с ключом 123 для базовой защиты от прямого чтения файла.

##  Анализ данных

Приложение вычисляет:
- Средний вес всех упражнений
- Максимальный вес (личный рекорд)
- Минимальный вес
- Общий объем тренировок (вес × повторения × подходы)
